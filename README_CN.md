# 船队三自由度模型 - Fleet 3DOF

## 项目简介

本项目实现了四船菱形编队的三自由度（3DOF）动力学模型，包含分布式路径规划、避障和控制系统。系统采用可变虚拟结构法保持编队队形，并能在遇到障碍物时平滑切换至独立避障模式。

## 核心技术

### 1. **三自由度船舶动力学模型** (Phase 1 ✅)
- 完整的ASV（自主水面船）动力学仿真
- 包含执行器延迟和输入限幅
- 水动力阻尼和耦合效应建模
- 支持环境扰动（风、流等）

**文件**: `Three_DOF_Model.py`

**主要类**:
- `ASV1`: 三自由度船舶动力学类
  - 状态量: [u, v, r, xn, yn, psin] （纵向速度、横向速度、艏向角速度、北向位置、东向位置、艏向角）
  - 控制输入: [tu, tr] （推力、转艏力矩）
  - 时间步长: 可配置（默认0.01s）

### 2. **GVF路径跟踪** (Phase 2 ✅)
- **引导向量场（Guiding Vector Field）**算法
- 圆形路径自动跟踪
- 切向+法向分量组合实现路径收敛

**文件**: `phase2_gvf_tracking.py`

**主要类**:
- `CirclePathGVF`: 圆形路径GVF生成器
  - 隐函数表示: φ(x,y) = (x-cx)² + (y-cy)² - R²
  - GVF向量: v = t - k_n·φ·n （t:切向，n:法向）
  - 法向增益k_n控制收敛速度

- `GVFController`: 基于GVF的航向控制器
  - 比例控制器: τ_r = k_heading × heading_error
  - 自动跟踪误差 < 1.5m

**测试结果**:
- ✅ 稳态平均误差: 1.38m
- ✅ 最大误差: 1.46m
- ✅ 误差标准差: 0.036m（无震荡）
- ✅ 400秒仿真无发散

### 3. **APF避障** (Phase 3 ✅)
- **人工势场法（Artificial Potential Field）**
- **环绕力场**：关键创新，避免局部极小值
- 切向力强度 > 引力强度，确保平滑绕行

**文件**: `phase3_apf_avoidance.py`

**主要类**:
- `APFController`: 势场法避障控制器
  - **引力场**: F_att = k_att × (goal - pos)
  - **斥力场**: F_rep = k_rep × d / dist³ （距离立方反比）
  - **环绕力场**: F_tan = k_tan × [-dy, dx] / dist （切向力，核心创新！）
  - 合力限幅保证稳定性

**关键参数**:
```python
k_att = 1.0   # 引力增益
k_rep = 8.0   # 斥力增益
k_tan = 12.0  # 切向力增益（必须 > k_att）
d_safe = 12.0 # 安全距离阈值
```

**测试结果**:
- ✅ 最小障碍距离: 1.74m（安全）
- ✅ 成功到达目标（误差4.98m）
- ✅ 轨迹平滑（最大步长0.1m）
- ✅ 184秒完成绕障

### 4. **GVF-APF混合控制** (Phase 4 ✅)
- **Sigmoid函数**实现平滑模态切换
- 自适应权重调整
- 无跳变的连续控制

**文件**: `phase4_hybrid_gvf_apf.py`

**主要类**:
- `HybridGVFAPFController`: 混合控制器
  - **权重函数**: α = sigmoid(d_obs - d_safe)
    - α ≈ 1: GVF路径跟踪模式（远离障碍）
    - α ≈ 0: APF避障模式（接近障碍）
  - **融合向量**: v = α·v_gvf + (1-α)·F_apf
  - 平滑过渡，无突变

**Sigmoid函数**:
```python
α(d) = 1 / (1 + exp(-k·(d - d_safe)))
```
- k: 陡峭度参数（默认0.3，较小值使过渡更平滑）
- d_safe: 安全距离阈值（15m）

**测试结果**:
- ✅ 最小障碍距离: 1.59m
- ✅ 路径跟踪误差: 5.14m（远离障碍时）
- ✅ GVF模式占比: 70.7%
- ✅ APF模式占比: 29.3%
- ✅ 权重最大变化: 0.0038（极其平滑）

## 技术特点

### ✨ 创新点

1. **环绕力场设计**
   - 传统APF易陷入局部极小值
   - 本项目引入切向力，沿障碍物边缘绕行
   - 切向力强度 > 引力，确保不卡住

2. **平滑模态切换**
   - Sigmoid函数实现连续权重调整
   - 避免控制输入跳变
   - 权重变化率 < 0.01/步

3. **自适应融合**
   - 根据障碍物距离实时调整策略
   - 远离障碍：专注路径跟踪
   - 接近障碍：自动避障
   - 无需人工干预

## 参数调优指南

### GVF参数
- `k_n`: 法向增益（0.5-2.0）
  - 过小：收敛慢
  - 过大：震荡

### APF参数
- `k_att`: 引力增益（0.5-2.0）
- `k_rep`: 斥力增益（5.0-15.0）
- `k_tan`: 切向力增益（10.0-20.0）
  - **关键**: k_tan > k_att，避免局部极小值
- `d_safe`: 安全距离（10-20m）

### 混合控制参数
- `k_sigmoid`: Sigmoid陡峭度（0.1-0.5）
  - 过小：切换慢
  - 过大：切换突然

### 航向控制参数
- `k_heading`: 航向增益（3.0-7.0）
- `max_tau_r`: 力矩限幅（1.0-2.0 N·m）
