整体需求：
四船菱形船队模型采用3DOF模型，分布式路径规划、避障和控制；
上层用可变的虚拟结构法保持船队队形，对虚拟结构本身进行空间映射变形，当遇到障碍物时，进入避障区域后，暂时解耦虚拟结构对各成员船的位姿约束，上层停止下发基于菱形几何关联的参考点，改为下发独立的任务目标点；
中层用GVF为每条船生成目标轨迹，APF做局部避障，在 APF 中加入环绕力场，引导船只沿障碍物边缘平滑绕行，并且切向绕行强度要高于GVF路径吸引，避障模式下，各船进入全自主导航模式，基于 Sigmoid 函数的加权机制不再受限于编队整体位置，而是由单船与障碍物的距离及相对速度决定，实现从‘编队协同路径跟随’向‘单船独立避障’的无缝模态切换；
下层用分布式NMPC控制器控制，在障碍物危险范围计算上引入一个自适应参数，当船速越快时，安全范围越大；
在避障模式下，在 NMPC 代价函数中引入时变队形权重矩阵Q，检测到障碍物入侵（自适应安全距离内）时，将队形保持权重Q设为 0 或极小值，使控制器专注于避障轨迹跟踪；
当检测到船只脱离障碍物危险区后，通过指数趋近函数平滑恢复Q的权重，引导船只回归由虚拟结构定义的参考锚点，NMPC输入端采用卡尔曼滤波增加对邻船状态的预测补偿，保证保持编队成员间的碰撞规避感知而非队形约束，在NMPC的代价函数中增加对低速状态的非线性惩罚，避免限入局部极小值。对APF生成的斥力向量场进行限幅，确保解耦后的独立避障轨迹始终满足非动力学约束与物理可行域。

关键名词介绍：
一、GVF（Guiding Vector Field）引导向量场
定位：路径跟踪 / 制导算法

    给轨迹周围造一个 “向量场”，车不管在哪，都被向量推着往轨迹靠
    优点：简单、稳定、收敛性好，适合无人车 / 无人机循迹
    缺点：不直接处理动力学约束，偏几何制导

二、NMPC（Nonlinear Model Predictive Control）非线性模型预测控制
定位：高级控制算法（轨迹跟踪 + 约束优化）

    用非线性车辆模型预测未来几步怎么走
    每一步在线求解优化：让跟踪误差最小，同时满足转向、车速、轮胎力、侧滑角等约束
    优点：精度高、鲁棒、能处理强非线性（轮胎、动力学）
    缺点：计算量大，需要好求解器

三、APF（Artificial Potential Field）人工势场法
定位：局部路径规划 / 避障算法

    目标是引力，障碍物是斥力，车沿势能最低方向走
    优点：极简单、实时性强
    缺点：容易局部极小值（卡住）、震荡、目标不可达

## Claude要写：

```python
class Ship3DOF:
    def __init__(self):
        self.x = 0
        self.y = 0
        self.psi = 0
        self.u = 1.0
        self.r = 0.0

    def step(self, tau_u, tau_r, dt):
        self.u += tau_u * dt
        self.r += tau_r * dt

        self.x += self.u * np.cos(self.psi) * dt
        self.y += self.u * np.sin(self.psi) * dt
        self.psi += self.r * dt
```

```python
def gvf_field(pos):
    x, y = pos
    grad = np.array([2*x, 2*y])

    t = np.array([-grad[1], grad[0]])   # 切向
    n = -grad                           # 法向

    return t + 0.3 * n
```

# 🚀 Phase 3：APF避障（关键突破）

## 🎯 目标

👉 船能绕开障碍物，而不是撞上去

---

## Claude实现：

```python
def apf_force(pos, goal, obs):
    d = pos - obs
    dist = np.linalg.norm(d)

    F_att = goal - pos

    if dist < 10:
        F_rep = d / (dist**3 + 1e-6)

        # ⭐ 环绕力场（核心）
        F_tan = np.array([-d[1], d[0]]) / (dist + 1e-6)
    else:
        F_rep = 0
        F_tan = 0

    return F_att + 2*F_rep + 3*F_tan
```
